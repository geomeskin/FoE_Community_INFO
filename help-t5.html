👥 Friend Manager

Tab 5 — Friends list with activity status, last-aided dates, and days since login

What it does

Tracks your friends list — who's active, who's gone quiet, when you last aided each friend, and how long they've been in your friends list. The main use is spotting inactive players to replace and maintaining reciprocal aiding habits.

What you'll see

  • One section per player (one section = one MegaExport zip).
  • Each row is one friend on that player's list.
  • Sort by Days Since to surface the most inactive friends at a glance.
  • Search by name or filter to Active / Inactive / New using the dropdowns.
  • Count at the top shows total friends and how many are inactive.

Column guide

ColumnMeaning
FriendIn-game name
Friend SinceDate you became friends
Last AidedDate you last aided this friend
Days SinceDays since this friend last logged in to FoE
StatusActive / Inactive / New (see below)

Status labels

Active Inactive New

Active — logged in recently.
Inactive — hasn't logged in for an extended period. These are the friends worth reviewing for replacement.
New — recently added friend, not yet established.

Data source & updates

Pulled from the friends section of each player's FoE Helper MegaExport zip.

Updated every Thursday when players run ./update.sh.

Days Since is a snapshot. The value reflects the friend's last login as recorded at the time of the export — it doesn't update in real time. A friend who logged in after the last export will still show their old "days since" count until the next update.
