⚔ Boost Reference

Tab 1 — Event building attack & defense boost values for every era

What it does

Displays every event building tracked in the guild's FoE Helper exports, with its attack and defense boost values scaled to whatever era you select. With 700+ buildings indexed, it's the most complete single-page boost reference available from Helper data.

All boost values are era-dependent — the same building can give very different numbers in Bronze Age vs Space Age. Selecting your era recalculates everything instantly.

How to use

  • Select your ERA from the gold dropdown — all values update instantly.
  • Type in the search box to filter by building name.
  • Use Type to narrow to event buildings, GbG-only boosts, or Guild Expedition buildings.
  • Use Event to show buildings from one specific event (e.g. Fall Event 2025).
  • Use Sort to order by Total, Per Tile, Att%, Def%, or GbG boosts.
  • Click any column header to sort by that column — click again to reverse.
  • The count badge on the right shows how many buildings match your current filters.

Data source

Built from the buildingMeta section inside each player's FoE Helper MegaExport zip. All exports are merged and deduplicated, so the widest possible building set is captured across all guild members' exports.

Updated weekly when players run ./update.sh.

Event timing matters. FoE Helper only includes event buildings in the MegaExport while that event is currently active in-game. Once an event ends, those buildings won't appear in new exports. If a building isn't showing up, it likely wasn't captured during the event window.
Pro tip: Export from the BETA server as soon as a new event launches there — BETA runs events ahead of live, so you can capture preview building data before the event goes live on your server.
