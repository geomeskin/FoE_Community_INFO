⚔ Battle Boosts

Tab 2 — Curated battle boost reference across 4 combat contexts · Data by Wyldfyre

What it does

A hand-curated list of the best battle boost buildings across four different combat contexts in FoE. Unlike Tab 1 which catalogs everything, this is an opinionated reference maintained by Wyldfyre (guild rank #2) — focused on the buildings actually worth prioritizing.

Each context has a different meta. A building that's excellent in Open World may be less useful in GBG. Switch contexts to see which buildings matter where.

The 4 contexts

Open World

Regular map combat and GvG. General-purpose attack and defense for your everyday army.

Guild Expedition

Weekly GEx encounters. GEx has its own boost values distinct from open world.

GBG

Guild Battlegrounds. High-intensity — GBG-specific boosts stack separately from open world.

QI

Quantum Incursion. A separate combat mode with its own boost framework.

Boost types explained

CodeFull nameWhat it does
AAAAttack for Attacking ArmyBoosts your offense when you attack
DAADefense for Attacking ArmyProtects your army while you're attacking
ADAAttack for Defending ArmyBoosts your city defense army's attack
DDADefense for Defending ArmyBoosts your city defense army's resilience

How to use

  • Select a context (Open World / GEx / GBG / QI) — values change per context.
  • Filter by source to see only Event buildings, GBG shop, QI shop, or Settlement rewards.
  • Sort by Total for overall power, or Per Tile for space efficiency.
  • Click any column header to sort — click again to reverse.

Data source & updates

This tab is driven by a CSV file maintained by Wyldfyre, not by FoE Helper exports. When Wyldfyre sends an updated file, drop it next to build.py with the name Wyldfyre_BattleBoosts_YYMMDD.csv and run ./update.sh. The script auto-detects any file matching *BattleBoost*.csv — no code changes needed.

Note: This list reflects Wyldfyre's judgment about which buildings are worth highlighting. It's not exhaustive — for a complete catalog of all event buildings use Tab 1 (Boost Reference).
